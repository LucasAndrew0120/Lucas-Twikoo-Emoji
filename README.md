# Lucas-Twikoo-Emoji

[Twikoo](https://github.com/imaegoo/twikoo) 表情包合集，包含百度贴吧经典表情、精华表情与小黄鸡贴纸，通过 jsDelivr CDN 加速。

## 使用方法

在 Twikoo 后台「自定义表情」中填入对应 JSON 文件的 Raw 链接：

- 贴吧经典：`https://cdn.jsdelivr.net/gh/LucasAndrew0120/Lucas-Twikoo-Emoji@main/Tieba-Classic.json`
- 贴吧高清：`https://cdn.jsdelivr.net/gh/LucasAndrew0120/Lucas-Twikoo-Emoji@main/Tieba-Refined.json`
- 小黄鸡：`https://cdn.jsdelivr.net/gh/LucasAndrew0120/Lucas-Twikoo-Emoji@main/xhj.json`
